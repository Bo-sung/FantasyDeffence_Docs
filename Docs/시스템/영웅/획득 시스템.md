## 투트랙(Two-track) 영입 시스템

영웅은 크게 **제조**와 **방문 영웅 영입** 두 가지 경로를 통해 획득할 수 있습니다.

---

### 1. 영웅 제조 (소녀전선 방식)

자원을 투입하여 정해진 시간 이후 결과물을 확인하는 방식입니다. 투입하는 자원의 종류와 양을 조절하여 특정 역할군이나 등급의 영웅을 저격할 수 있습니다.

#### 핵심 요소

| 항목 | 내용 |
| --- | --- |
| **필요 재화** | **전투 자원 4종** (식량, 마석, 보급, 인력) + **모집권 1개** |
| **슬롯** | 기본 2개 제공, **금화**를 사용하여 최대 6개까지 확장 가능 |
| **시간** | 투입된 레시피와 결과물 등급에 따라 제조 완료까지 일정 시간 소요 |
| **픽업** | 특정 기간 동안 특정 영웅의 제조 확률이 증가하는 **픽업 이벤트** 적용 |

#### 레시피 시스템

제조 시 투입하는 4종의 전투 자원 **비율**과 **총합**에 따라 결과가 달라집니다.

- **자원 비율**: 결과로 나올 영웅의 **역할군(클래스)** 에 영향
  - 예: `식량` 비율이 높으면 탱커, `마석` 비율이 높으면 마법사 등장 확률 증가
- **자원 총합**: 결과로 나올 영웅의 **기본 등급**에 영향
  - 예: 자원 총합이 높을수록 높은 등급(4~5성)의 영웅 등장 확률 증가

> 상세 레시피는 별도 문서로 관리합니다.

---

### 2. 방문 영웅 영입

일정 시간마다 영지에 방문하는 영웅을 **확정적으로 영입**하는 방식입니다.

#### 핵심 요소

| 항목         | 내용                                            |
| ---------- | --------------------------------------------- |
| **방문 주기**  | 24시간마다 초기화                                    |
| **방문 영웅**  | 매 주기마다 **랜덤한 영웅 1명**이 영지에 방문                  |
| **필요 재화**  | **금화** (결제 재화)                                |
| **영입 방식**  | 정해진 금화를 지불하고 **즉시 확정 영입**                     |
| **랜덤 풀**   | **픽업 이벤트와 무관**하게 전체 영웅 목록에서 랜덤으로 등장           |
| **놓쳤을 경우** | 영입하지 않고 주기가 지나가면 해당 영웅은 사라지며, 다시 등장하기까지 기약 없음 |

# 시스템 기획서

## 2.1 게임 개요

**게임명**: (미정) **장르**: 타워디펜스 RPG **플랫폼**: 모바일 (iOS/Android) **화면 방향**: 세로 (Portrait) **시점**: 쿼터뷰 (탑뷰 개념) **타겟**: 서브컬처 게임 유저, 타워디펜스 장르 선호층

---

## 2.2 핵심 게임 루프

### 메타 루프 (중장기)

```
1. 영지 관리 (자원 생산, 시설 업그레이드)
   ↓
2. 영웅 영입 (주점 시스템)
   ↓
3. 영웅 육성 (레벨업, 링크 확장, 장비)
   ↓
4. 스테이지 도전 (전투)
   ↓
5. 보상 획득 (재료, 골드, 경험치)
   ↓
6. 영지 강화 (성벽 무기 업그레이드 등)
   ↓
7. 반복 (더 어려운 스테이지 도전)
```

### 전투 루프 (단기)

```
1. 덱 편성 (영웅 선택, 방어 시설 준비)
   ↓
2. 스테이지 진입
   ↓
3. 준비 단계 (방어 시설 배치)
   ↓
4. 웨이브 시작
   ↓
5. 실시간 전투 (영웅 부대 vs 마물)
   ↓
6. 웨이브 클리어 또는 실패
   ↓
7. 다음 웨이브 또는 스테이지 클리어
   ↓
8. 보상 획득
   ↓
9. 수리 (손상된 영웅 회복)
```

## 2.3 전투 시스템

### 2.3.1 맵 구조

**화면 레이아웃 (세로 화면)**

```
┌─────────────────────┐
│   [상단: 산맥]       │ ← 통과 불가 지형
├─────────────────────┤
│                     │
│  [우측: 적 출현]    │ ← 마물 스폰 영역
│        ↓ ↓ ↓        │
│                     │
│  [중앙: 전투장]     │ ← 주 전투 공간
│   방어시설 배치     │
│   영웅 부대 활동    │
│                     │
│  [좌측: 성벽]       │ ← 최종 방어선
│   ████████████     │    (발리스타, 투석기)
│                     │
├─────────────────────┤
│   [하단: 산맥]       │ ← 통과 불가 지형
└─────────────────────┘
```

**시점**: 쿼터뷰

- 위에서 대각선으로 내려다보는 시점
- 개념상 탑뷰로 이해 가능
- 깊이감 표현하되 픽셀아트로 단순화

**맵 영역 구분**

- **성벽 영역**: 좌측 끝, 성벽 무기 고정 배치
- **전투 영역**: 성벽과 적 출현 사이, 넓은 공간
- **적 출현 영역**: 우측, 마물 스폰 지점
- **산맥**: 상하단, 통과 불가 (경계 설정)

### 2.3.2 준비 단계

**방어 시설 배치**

플레이어는 제한된 **건설 포인트** 내에서 방어 시설 배치

**건설 포인트 시스템**

- 스테이지마다 주어지는 배치 예산
- 예: 초급 스테이지 100pt, 고급 스테이지 200pt
- 시설별 비용이 다름

**배치 가능 시설**

|시설명|비용|효과|특징|
|---|---|---|---|
|목책|20pt|적 이동 차단|높은 내구도, 병목 생성|
|함정|30pt|데미지 + 상태이상|1회용, 둔화/출혈/스턴|
|화살탑|50pt|자동 공격 (물리)|중거리, 단일 타겟|
|마법탑|60pt|자동 공격 (마법)|범위 공격|
|보급소|40pt|아군 지원|HP 회복 또는 버프|

**배치 규칙**

- 그리드 기반 배치 (타일 단위)
- 일부 타일은 배치 불가 (경로 확보)
- 시설 간 최소 거리 제한 가능
- 전략적 배치로 적 경로 통제

**배치 UI**

```
┌─────────────────────┐
│ 건설 포인트: 100/100 │
├─────────────────────┤
│ [목책] [함정] [화살탑] │
│  20pt   30pt   50pt  │
│                     │
│ [마법탑] [보급소]    │
│  60pt    40pt       │
├─────────────────────┤
│    [배치 완료]       │
└─────────────────────┘
```

### 2.3.3 전투 단계

**웨이브 시스템**

각 스테이지는 여러 웨이브로 구성

- 예: 스테이지 1-1 = 웨이브 3개
- 예: 스테이지 5-10 = 웨이브 7개

**웨이브 진행**

1. 웨이브 시작 알림 (5초 카운트다운)
2. 적 유닛 우측에서 출현
3. 적 유닛 성벽을 향해 이동
4. 영웅 부대 자동 교전
5. 성벽 무기 사거리 내 적 자동 공격
6. 모든 적 처치 시 웨이브 클리어
7. 다음 웨이브 대기 (10초)

**적 행동 패턴**

- 기본: 성벽을 향해 직진
- 변형: 우회, 빠른 돌파, 비행 등
- 목표: 성벽 도달 → 성벽 공격

**실시간 조작**

플레이어는 전투 중 제한적 명령 가능:

|명령|효과|쿨타임|
|---|---|---|
|후퇴|선택 영웅 부대 후방 이동|20초|
|집중 공격|특정 적 우선 타겟팅|10초|
|위치 고정|현 위치 사수|즉시|
|스킬 발동|영웅 고유 스킬 수동 사용|스킬별|

**승리 조건**

- 모든 웨이브의 적 전멸

**패배 조건**

- 성벽 내구도 0 도달
- 또는 모든 영웅 부대 전멸

### 2.3.4 성벽 무기 시스템

**고정 방어 무기**

성벽 위에 설치된 자동 공격 무기

- 영지에서 미리 업그레이드
- 스테이지마다 동일 성능 적용
- 배치 불필요 (항상 활성화)

**무기 종류**

**발리스타 (Ballista)**

- 타입: 단일 고화력 물리 공격
- 사거리: 중거리
- 공격속도: 느림 (3초)
- 특성: 관통 가능
- 업그레이드: 공격력 ↑, 장전 속도 ↑, 관통 수 ↑

**투석기 (Catapult)**

- 타입: 범위 공격
- 사거리: 장거리
- 공격속도: 매우 느림 (5초)
- 특성: 착탄 지점 중심 AoE
- 업그레이드: 데미지 ↑, 범위 ↑, 재장전 ↑

**마법 포대 (Magic Cannon)**

- 타입: 속성 공격 (화염/빙결/번개)
- 사거리: 중장거리
- 공격속도: 중간 (2초)
- 특성: 상태이상 부여
- 업그레이드: 속성 효과 ↑, 연사 ↑, 범위 ↑

**성벽 내구도**

- 스테이지 시작 시 100% (또는 영지 업그레이드 수치)
- 적이 성벽 도달 시 공격받음
- 내구도 0 = 즉시 패배
- 영지 관리로 최대 내구도 증가 가능

## 2.4 덱 편성 시스템

### 2.4.1 기본 구조

**코스트 없음**

- 명일방주식 코스트 시스템 사용 안 함
- 사전에 편성한 덱이 전투에 모두 참여
- 전략은 "누구를 데려갈 것인가"에 집중

**덱 슬롯**

- 기본: 5슬롯
- 확장: 플레이어 레벨 또는 업그레이드로 최대 8슬롯

**편성 화면**

```
┌─────────────────────────┐
│    덱 편성 (5/5)         │
├─────────────────────────┤
│ [슬롯1] [슬롯2] [슬롯3]  │
│ 기사    궁수    마법사   │
│ Lv.30   Lv.25   Lv.28   │
│ 5링크   4링크   3링크    │
│                         │
│ [슬롯4] [슬롯5]  [빈슬롯] │
│ 힐러    돌격병    ---    │
│ Lv.22   Lv.27           │
│ 3링크   5링크           │
├─────────────────────────┤
│  [영웅 변경] [출격]      │
└─────────────────────────┘
```

### 2.4.2 영웅 역할 분류

게임 내에서 영웅은 역할군으로 구분

|역할|특징|병종 예시|
|---|---|---|
|탱커|높은 HP/방어, 전방 방어|중보병, 기사|
|딜러|높은 공격력, 적 처치|검사, 중궁병|
|원거리|중후방 지원 공격|궁수, 마법사|
|지원|아군 회복/버프|성직자, 연금술사|
|특수|고유 메커니즘|암살자, 소환사|

**역할 간 시너지**

- 탱커가 전방에서 적 묶음
- 딜러가 주 화력 담당
- 원거리가 안전한 위치에서 지원
- 지원이 생존력 제공

## 2.5 링크 시스템 (더미 시스템)

### 2.5.1 개념

**소녀전선 더미 링크 차용**

- 영웅 1명이 여러 병사를 지휘
- 병사 = 영웅의 확장 체력/화력
- 링크 수 = 부대 전투력

**시각적 표현**

```
5링크 부대:
[영웅] [병사] [병사] [병사] [병사]

3링크 부대 (일부 손실):
[영웅] [병사] [병사]

1링크 (영웅만 생존):
[영웅]
```

### 2.5.2 링크 구조

**링크 단계**

- 1링크: 영웅만 (기본)
- 2링크: 영웅 + 병사 1명
- 3링크: 영웅 + 병사 2명
- 4링크: 영웅 + 병사 3명
- 5링크: 영웅 + 병사 4명 (최대)

**스탯 증가 공식**

- 1링크: 기본 스탯 100%
- 2링크: 기본 스탯 180%
- 3링크: 기본 스탯 260%
- 4링크: 기본 스탯 340%
- 5링크: 기본 스탯 420%

**링크별 화력**

- 각 링크는 개별 공격 가능
- 5링크 = 5배 DPS (단순 계산)
- 실제로는 스킬/버프 등 변수 존재

### 2.5.3 링크 확장 방법

**레벨 조건**

|링크|필요 레벨|추가 조건|
|---|---|---|
|2링크|Lv.10|재료 소모|
|3링크|Lv.30|재료 소모|
|4링크|Lv.70|재료 소모|
|5링크|Lv.90|재료 소모|

**재료 옵션**

1. **동일 영웅 소모**
    - 같은 영웅 1명을 재료로 사용
    - 중복 뽑기 활용
2. **범용 코어 사용**
    - 일반 코어 (2링크용)
    - 고급 코어 (3링크용)
    - 희귀 코어 (4링크용)
    - 전설 코어 (5링크용)

**코어 획득 방법**

- 일일 던전 드롭
- 이벤트 보상
- 상점 구매 (골드 또는 보석)
- 중복 영웅 분해

### 2.5.4 전투 중 링크 손실

**피해 메커니즘**

부대 HP = 영웅 HP + (병사 HP × 병사 수)

예시 (5링크 부대):

- 영웅 HP: 1000
- 병사 HP (각): 500
- 총 HP: 1000 + (500 × 4) = 3000

**손실 단계**

|남은 HP|링크 상태|화력|상태|
|---|---|---|---|
|100~80%|5링크|100%|정상|
|80~60%|4링크|80%|병사 1 손실|
|60~40%|3링크|60%|병사 2 손실|
|40~20%|2링크|40%|병사 3 손실|
|20~1%|1링크|20%|영웅만 생존|
|0%|전사|0%|부대 전멸|

**손실 순서**

- 병사부터 먼저 손실 (영웅 보호)
- 영웅은 마지막까지 생존
- HP 구간 진입 시 즉시 링크 감소

**시각 효과**

- 링크 손실 시 병사 유닛 화면에서 사라짐
- 남은 병사만 표시
- HP 바 색상 변화 (녹→황→적)

### 2.5.5 자동 퇴각 시스템

**퇴각 조건**

- HP 20% 이하 도달 시 자동 퇴각

**퇴각 프로세스**

1. HP 20% 이하 감지
2. 자동으로 후방 이동 시작
3. 성벽 뒤로 이동 완료
4. 해당 스테이지 내 재출격 불가
5. 스테이지 종료 후 자동 회복

**퇴각의 의미**

- 영웅 전사 방지 (수리 비용 절감)
- 전술적 후퇴 허용
- 하지만 전력 감소로 스테이지 클리어 어려워질 수 있음

### 2.5.6 전사 시스템

**전사 조건**

- HP 0 도달

**전사 시 영향**

- 해당 스테이지에서 완전 제외
- 재출격 불가
- 스테이지 실패 시에도 영구 손실 없음

**전사 후 처리**

- 스테이지 성공/실패와 무관하게 전사 처리
- 수리실에서 수리 필요 (높은 비용)

## 2.6 수리 시스템

### 2.6.1 수리 개념

**전투 후 관리**

- 전투에서 손상된 링크는 수리 필요
- 수리하지 않으면 다음 전투에 손상 상태로 참전
- 소녀전선과 동일한 메커니즘

### 2.6.2 수리 비용

**비용 구성**

- 골드 (기본 화폐)
- 인력 (영지에서 생산하는 자원)

**손실 정도별 비용**

|손실|골드|인력|수리 시간|
|---|---|---|---|
|1링크 손실|100|1|10분|
|2링크 손실|200|2|30분|
|3링크 손실|350|4|1시간|
|4링크 손실|500|7|2시간|
|영웅 전사|1000|10|4시간|

**보석으로 즉시 수리**

- 즉시 완료 옵션 제공
- 보석 소모량은 남은 시간에 비례

### 2.6.3 수리실 시스템

**수리 슬롯**

- 기본: 2슬롯 (동시 2명 수리 가능)
- 확장: 보석 또는 영지 업그레이드로 최대 4슬롯

**수리 대기열**

- 슬롯 초과 시 대기
- 순서대로 자동 진입

**수리 UI**

```
┌─────────────────────────┐
│       수리실             │
├─────────────────────────┤
│ [슬롯1] 영웅A 수리 중    │
│  남은 시간: 45분         │
│  [즉시 완료: 50 보석]    │
│                         │
│ [슬롯2] 영웅B 수리 중    │
│  남은 시간: 1시간 30분   │
│  [즉시 완료: 90 보석]    │
├─────────────────────────┤
│ 대기: 영웅C, 영웅D       │
└─────────────────────────┘
```

### 2.6.4 전투 전 보급

**자동 보급**

- 전투 시작 전 모든 링크 100% 상태
- 별도 보급 과정 불필요
- 단, 수리 미완료 시 손상 상태로 출격

## 2.7 영웅 수급 시스템

### 2.7.1 주점 시스템

**컨셉**

- 가챠가 아닌 "주점에서 영웅 만남"
- 세계관 친화적 영입 방식

**주점 화면**

```
┌─────────────────────────┐
│   🍺 변방의 주점 🍺      │
├─────────────────────────┤
│ [일반 영입]  [프리미엄]  │
│  100 골드    300 보석   │
│                         │
│ [오늘의 특별 손님]       │
│  ⭐⭐⭐ 베테랑 기사    │
│  [영입: 500 골드]       │
└─────────────────────────┘
```

**영입 방식**

1. **일반 영입**
    
    - 비용: 100 골드
    - 등급: 1~3성 위주
    - 가끔 4성 출현
2. **프리미엄 영입**
    
    - 비용: 300 보석
    - 등급: 2~5성
    - 4~5성 확률 높음
3. **특별 손님**
    
    - 매일 1명씩 등장
    - 고정 비용으로 직접 영입 가능
    - 놓치면 다시 등장까지 시간 소요
4. **의뢰 보상**
    
    - 특정 스테이지 클리어 보상
    - 특정 영웅 확정 획득

### 2.7.2 등급 시스템

**별 등급**

- 1성 ~ 5성

**중요: 모든 영웅 5성까지 승급 가능**

- 1성도 육성하면 5성 도달
- 초기 등급은 획득 난이도와 초반 성능에만 영향
- 최종 성능은 모두 동일 레벨에 수렴

**등급별 특징**

|등급|초기 능력치|획득 난이도|육성 가치|
|---|---|---|---|
|1성|낮음|매우 쉬움|장기 프로젝트|
|2성|보통|쉬움|중기 육성|
|3성|보통+|보통|단기 육성|
|4성|높음|어려움|즉시 전력화|
|5성|매우 높음|매우 어려움|즉시 주력|

## 2.8 육성 시스템

### 2.8.1 레벨링

**경험치 획득**

- 전투 참여 시 획득
- 경험치 아이템 사용

**레벨 상한**

- 1성: Lv.30 → 승급 후 Lv.50
- 2성: Lv.40 → 승급 후 Lv.60
- ...
- 5성: Lv.100 (최종)

### 2.8.2 승급 (Promotion)

**승급 시스템**

- 특정 레벨 도달 + 재료 소모
- 성급 상승 (1성 → 2성 → ... → 5성)
- 스탯 대폭 증가
- 스킬 강화 또는 해금
- 외형 변화 (3성, 5성에서 주로)

**승급 재료**

- 범용 재료 (던전 파밍)
- 등급별 특수 재료
- 골드

**5성 도달 특전**

- 최종 일러스트 해금
- Live2D 적용 (선택적)
- 고유 스킨 획득

### 2.8.3 스킬 시스템

**스킬 종류**

- 액티브 스킬: 수동 발동 또는 자동 (쿨타임)
- 패시브 스킬: 항상 적용

**스킬 레벨업**

- 스킬북 아이템 소모
- 레벨당 효과 강화 (데미지, 지속시간 등)
- 최대 Lv.10

**스킬 해금**

- 승급 시 새로운 스킬 해금
- 예: 3성 승급 시 패시브 추가, 5성 승급 시 궁극기 해금

### 2.8.4 장비 시스템

**장비 슬롯**

- 무기: 공격력 증가
- 방어구: 방어력/HP 증가
- 액세서리: 특수 효과 (치명타, 회피 등)

**장비 획득**

- 영지 대장간에서 제작
- 스테이지 드롭
- 이벤트 보상

**장비 강화**

- 강화 재료 + 골드 소모
- 최대 +10 강화
- 높은 강화 수치일수록 실패 확률 존재 (선택적)

|시설|기능|
|---|---|
|주점|영웅 영입|

**편의 시설**

|시설|기능|
|---|---|
|숙소|영웅 배치 시 신뢰도 상승, 소량 재화|
|식당|영웅 간 교류 이벤트|
|광장|이벤트 장소|

### 2.9.3 시설 운영

**시설 레벨업**

- 건설 재료 + 골드 소모
- 생산량/효율 증가
- 새로운 기능 해금

**영웅 배치**

- 시설에 영웅 배치로 보너스
- 특정 영웅은 특정 시설에 특화 보너스
- 배치 중 영웅은 전투 불가

**자동 생산**

- 시간당 일정량 자동 생산
- 최대 저장량 존재 (넘치면 손실)
- 정기적 수확 필요

### 2.9.4 영지와 전투의 연결

**성벽 무기**

- 무기고 레벨 = 성벽 무기 성능
- 모든 전투에 영향

**방어 시설**

- 공병 작업장에서 사전 제작
- 전투 시 소모

**인력 자원**

- 훈련소에서 생산
- 링크 확장/수리 필수 자원

## 재화 목록

| 재화 | 분류 | 재생 방식 | 주요 용도 |
|------|------|-----------|-----------|
| 은화 | 범용 | 영지 세금 | 강화, 건설 |
| 금화 | 프리미엄 | 재생 없음 | 방문 영웅 영입, 즉시 완료 |
| 식량 | 전투 자원 | 영지 생산 | 부대 출격, 영웅 제조 |
| 마석 | 전투 자원 | 영지 생산 | 마법 포대 출격, 영웅 제조 |
| 보급 | 전투 자원 | 영지 생산 | 물리 포탑 출격, 소모품, 영웅 제조 |
| 인력 | 영지 자원 | 영지 생산 | 링크 확장/수리, 영웅 제조 |
| 모집권 | 영입 자원 | 영지 생산 (주점) | 영웅 제조 |

---

## 공통 요소: 만족도

영지 내 모든 생산에 적용되는 배율

```
만족도 = 50% (기본) + 편의 시설 보너스
범위: 50% ~ 150%
```

| 편의 시설 | 만족도 보너스 |
|-----------|---------------|
| 우물 | +5% |
| 방앗간 | +10% |
| 교회 | +15% |
| 광장 | +10% |
| 시장 | +20% |

## 3. 식량

### 생성 공식

```
시간당 생산 = 기본 생산량 × 만족도
```

### 농장 생산량

| 레벨 | 기본 생산량 | 최대 저장량 |
|------|-------------|-------------|
| 1 | 60 | 500 |
| 2 | 90 | 750 |
| 3 | 130 | 1,000 |
| 4 | 180 | 1,500 |
| 5 | 250 | 2,000 |

### 소모 공식 (출격)

```
출격 소모 = 기본 소모량 × 링크 배율
```

| 링크 수 | 링크 배율 |
|---------|-----------|
| 1링크 | 1.0 |
| 2링크 | 1.5 |
| 3링크 | 2.0 |
| 4링크 | 2.5 |
| 5링크 | 3.0 |

| 영웅 등급 | 기본 소모량 |
|-----------|-------------|
| 1성 | 5 |
| 2성 | 8 |
| 3성 | 12 |
| 4성 | 18 |
| 5성 | 25 |

### 소모 예시

```
5성 영웅 5링크 출격:
식량 소모 = 25 × 3.0 = 75 식량
```

### 소모 (영웅 제조)

제조 레시피에 따라 투입 (30~999)

```
수리 소모 = 손실 링크 × 링크당 수리 비용
```

| 영웅 등급 | 링크당 수리 비용 |
|-----------|------------------|
| 1성 | 2 |
| 2성 | 3 |
| 3성 | 5 |

### 소모 예시

```
5성 영웅 2링크 손실 수리:
```

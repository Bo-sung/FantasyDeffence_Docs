[← README로 돌아가기](../README.md)

## 재화 개요

| 재화 | 분류 | 주요 용도 |
|------|------|-----------|
| 은화 | 범용 | 강화, 건설 |
| 금화 | 프리미엄 | 방문 영웅 영입, 즉시 완료, 슬롯 확장 |
| 식량 | 전투 자원 | 부대 출격, 영웅 제조 |
| 마석 | 전투 자원 | 마법 포대 출격, 영웅 제조 |
| 보급 | 전투 자원 | 물리 포탑, 소모품, 영웅 제조 |
| 인력 | 영지 자원 | 링크 확장/수리, 영웅 제조 |
| 모집권 | 영입 자원 | 영웅 제조 |

---

## 범용 재화

### 은화

일반적인 가챠게임의 골드와 같은 역할

| 항목 | 내용 |
|------|------|
| 획득 | 스테이지 클리어, 영지 세금, 일일 던전, 영웅 분해 |
| 용도 | 장비 강화, 건물 건설/업그레이드, 구역 확장, 스킬 강화 |
| 특징 | 다양한 곳에서 획득, 다양한 곳에 소모 |

### 금화

가챠게임의 보석/주얼과 같은 프리미엄 재화

| 항목 | 내용 |
|------|------|
| 획득 | 스테이지 첫 3성 클리어, 이벤트, 퀘스트, 과금 |
| 용도 | 방문 영웅 영입, 제조 슬롯 확장, 즉시 완료, 스태미나 회복 |
| 특징 | 제한된 획득처, 신중한 사용 유도 |

---

## 전투 자원

출격 및 영웅 제조 시 소모되는 자원 (칸코레/소녀전선 스타일)

### 식량

| 항목 | 내용 |
|------|------|
| 획득 | 영지 농장 생산 |
| 용도 | 부대 출격 (링크 비례), **영웅 제조** |
| 특징 | 제조 레시피에서 비율 높으면 탱커 확률 증가 |

### 마석

| 항목 | 내용 |
|------|------|
| 획득 | 마력 정제소, 던전 보상 |
| 용도 | 마법 포대 출격, **영웅 제조** |
| 특징 | 제조 레시피에서 비율 높으면 마법사 확률 증가 |

### 보급

| 항목 | 내용 |
|------|------|
| 획득 | 영지 보급창고, 스테이지 보상 |
| 용도 | 발리스타/투석기 출격, 목책 제작, **영웅 제조** |
| 특징 | 물리 계열 무기, 소모품, 제조 레시피 영향 |

## 영지 자원

### 인력

| 항목 | 내용 |
|------|------|
| 획득 | 훈련소 생산 |
| 용도 | 링크 확장, 링크 수리, **영웅 제조** |
| 특징 | 병사 충원 개념, 영웅 성장의 핵심 자원 |

### 모집권

| 항목 | 내용 |
|------|------|
| 획득 | 주점 생산 (1일 1개), 상점 구매 (금화 100), 이벤트 |
| 용도 | **영웅 제조** (전투자원 4종 + 모집권 1개) |
| 특징 | 제조 시 필수 소모, 무과금 플레이 페이스 조절 |

## 출격 비용 체계

### 영웅 부대

| 링크 수 | 식량 소모 |
|---------|-----------|
| 1링크 | 기본 |
| 3링크 | 중간 |
| 5링크 | 높음 |

### 성벽 무기

| 무기 | 소모 자원 | 비용 |
|------|-----------|------|
| 발리스타 | 보급 | 낮음 |
| 투석기 | 보급 | 중간 |
| 마법 포대 | 마석 | 높음 |

## 재화 밸런싱

### 무과금 기준

| 항목 | 목표 |
|------|------|
| 일일 은화 | 충분히 수리/강화 가능 |

## 엔진

### 추천: Unity

- 2D 스프라이트 처리 우수
- 픽셀아트 파이프라인 성숙
- 모바일 최적화 노하우

### 대안: Unreal Engine

- 기존 UE5 경험 활용
- Paper2D 또는 커스텀 2D 시스템

## 픽셀 퍼펙트 고려사항

### 해상도

- 기준: 1920×1080 (가로)
- Pixels Per Unit: 16 or 32
- 필터: Point (No Filter)

### UI 스케일

- 픽셀아트: 선명한 스케일링
- 텍스트: 고해상도 폰트
- 아이콘: 픽셀 또는 벡터

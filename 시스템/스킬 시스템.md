# 스킬 시스템

[← 전투 메카닉](전투%20메카닉.md)

---

## 개요

영웅이 사용하는 능동/수동 스킬의 구조와 발동 조건을 정의합니다.

---

## 스킬 분류

### 발동 방식

| 분류 | 설명 | 예시 |
|------|------|------|
| 액티브 | 수동 발동, 쿨다운 존재 | 강타, 파이어볼 |
| 패시브 | 항상 적용, 조건부 발동 | 공격력 증가, 피격 시 반격 |
| 오토 | 조건 충족 시 자동 발동 | HP 50% 이하 시 회복 |

### 대상 범위

| 범위 | 설명 |
|------|------|
| 단일 | 지정한 적 1체 |
| 범위 | 지정 위치 반경 내 모든 적 |
| 직선 | 발사 방향 직선상 모든 적 |
| 자신 | 시전자 자신 |
| 아군 | 지정한 아군 1체 또는 범위 |

### 피해 타입

| 타입 | 적용 방어력 |
|------|-------------|
| 물리 | DEF |
| 마법 | RES |
| 고정 | 방어력 무시 |
| 회복 | - (HP 회복) |

## 스킬 구조

### 기본 속성

```
skill {
  id           -- 스킬 고유 ID
  name         -- 스킬명
  type         -- active / passive / auto
  damage_type  -- physical / magical / fixed / heal
  target_type  -- single / area / line / self / ally
  range        -- 사거리
  cooldown     -- 쿨다운 (초)
  cost         -- 소모 자원 (있을 경우)
}
```

### 피해 공식

```
스킬 피해량 = 기본값 + (스탯 × 계수)
```

→ [전투 메카닉 - 데미지 계산 공식](전투%20메카닉.md#데미지-계산-공식) 참조

## 쿨다운 시스템

### 쿨다운 타입

| 타입 | 설명 |
|------|------|
| 시간 기반 | X초 후 재사용 가능 |
| 턴 기반 | X회 공격 후 재사용 (미사용) |

### 쿨다운 감소

- 특정 장비/버프로 쿨다운 감소 가능
- 최소 쿨다운: 원본의 50%

---

## 스킬 강화

### 강화 방식

| 방식 | 효과 | 재료 |
|------|------|------|
| 레벨업 | 피해량/효과 증가 | 스킬북 + 은화 |

### 강화 효과 (레벨당)

- 피해 계수 +5~10%
- 쿨다운 -0.5초 (특정 레벨)
- 추가 효과 해금 (특정 레벨)

---

## 스킬 예시

### 물리 딜러 (근접)

| 스킬 | 타입 | 공식 | 쿨다운 | 효과 |
|------|------|------|--------|------|
| 강타 | 액티브 | ATK × 1.5 | 8초 | 단일 대상 강력한 일격 |
| 연속베기 | 액티브 | ATK × 0.8 × 3회 | 12초 | 3연속 공격 |
| 전투 본능 | 패시브 | - | - | HP 50% 이하 시 ATK +20% |

### 마법사 (원거리)

| 스킬 | 타입 | 공식 | 쿨다운 | 효과 |
|------|------|------|--------|------|
| 파이어볼 | 액티브 | 50 + MAG × 2.0 | 10초 | 범위 마법 피해 |
| 빙결 | 액티브 | MAG × 1.2 | 15초 | 단일 대상, 이동속도 50% 감소 3초 |
| 마력 증폭 | 패시브 | - | - | MAG +15% |

### 힐러 (지원)

| 스킬 | 타입 | 공식 | 쿨다운 | 효과 |
|------|------|------|--------|------|
| 치유의 빛 | 액티브 | MAG × 1.5 | 12초 | 아군 단일 HP 회복 |
| 보호막 | 액티브 | MAG × 1.0 | 20초 | 아군 단일 보호막 부여 |
| 축복 | 패시브 | - | - | 주변 아군 HP 재생 +5% |

## 발동 우선순위 (오토 스킬)

1. 궁극기 (조건 충족 시)
2. 쿨다운 완료된 액티브 스킬
3. 기본 공격

※ 수동 조작 모드에서는 플레이어가 직접 선택

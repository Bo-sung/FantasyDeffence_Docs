## 메인 스토리 (캠페인)

### 구조

- 챕터 구성 (8-12 스테이지/챕터)
- 난이도: 일반/하드/악몽

### 보상

| 조건 | 보상 |
|------|------|
| 첫 클리어 | 보석, 영웅 |
| 3성 달성 | 추가 보상 |
| 난이도별 | 별도 보상 |

### 별 평가 조건 예시

- 성벽 내구 80% 이상 유지
- 영웅 손실 1링크 이하
- 3분 내 클리어

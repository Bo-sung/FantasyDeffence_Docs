## 핵심 게임 루프

### 메타 루프 (중장기)

```mermaid
flowchart TD
    A[영지 관리] --> B[영웅 영입]
    B --> C[영웅 육성]
    C --> D[스테이지 도전]
    D --> E[보상 획득]
    E --> F[영지 강화]
    F --> A
```

### 전투 루프 (단기)

```mermaid
flowchart TD
    A[덱 편성] --> B[스테이지 진입]
    B --> C[준비 단계]
    C --> D[웨이브 시작]
    D --> E[실시간 전투]
    E --> F{클리어?}
    F -->|성공| G[다음 웨이브]
    F -->|실패| H[재도전]
    G --> I{마지막?}
    I -->|아니오| D
    I -->|예| J[보상 획득]
    J --> K[수리]
```

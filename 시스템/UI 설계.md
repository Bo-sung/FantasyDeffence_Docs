## 메인 화면 구성

```mermaid
flowchart TB
    subgraph 메인["메인 화면"]
        direction TB
        상단["상단: 플레이어 정보 / 재화"]
        중앙["중앙: 영지 화면"]
        하단["하단: 메뉴 탭"]
    end

    하단 --> 전투[전투]
    하단 --> 영지[영지]
    하단 --> 주점[주점]
    하단 --> 영웅[영웅]
    하단 --> 임무[임무]
    하단 --> 상점[상점]
```

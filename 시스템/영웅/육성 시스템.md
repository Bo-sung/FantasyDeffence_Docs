- 경험치 아이템 사용

| 등급 | 레벨 상한 | 승급 후 상한 |

| 1성 | Lv.30 | Lv.50 |
| 2성 | Lv.40 | Lv.60 |

## 승급 (Promotion)

### 승급 시스템

- 특정 레벨 도달 + 재료 소모
- 성급 상승 (1성 → 2성 → ... → 5성)
- 스탯 대폭 증가
- 스킬 강화 또는 해금
- 외형 변화 (3성, 5성에서 주로)

### 승급 재료

- 범용 재료 (던전 파밍)
- 등급별 특수 재료
- 골드

### 5성 도달 특전

- 최종 일러스트 해금
- Live2D 적용 (선택적)
- 고유 스킨 획득

## 스킬 시스템

### 스킬 종류

| 종류 | 설명 |
|------|------|
| 액티브 스킬 | 수동 발동 또는 자동 (쿨타임) |
| 패시브 스킬 | 항상 적용 |

### 스킬 레벨업

- 스킬북 아이템 소모
- 레벨당 효과 강화 (데미지, 지속시간 등)
- 최대 Lv.10

### 스킬 해금

| 승급 | 해금 내용 |
|------|-----------|
| 1성 | 기본 액티브 스킬 |
| 3성 | 패시브 스킬 추가 |
| 5성 | 궁극기 해금 |

## 장비 시스템

### 장비 슬롯

| 슬롯 | 주요 효과 |
|------|-----------|
| 무기 | 공격력 증가 |
| 방어구 | 방어력/HP 증가 |
| 액세서리 | 특수 효과 (치명타, 회피 등) |

### 장비 획득

- 영지 대장간에서 제작
- 스테이지 드롭
- 이벤트 보상

### 장비 강화

- 강화 재료 + 골드 소모
- 최대 +10 강화
- 높은 강화 수치일수록 실패 확률 존재 (선택적)
